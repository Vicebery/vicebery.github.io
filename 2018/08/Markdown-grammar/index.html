缩进

  不管是在段首还是在文章中间，都可以用以下几种方式实现word写作中的“空格”，也就算缩进。
  半方大的空白&ensp;或&#8194;
  全方大的空白&emsp;或&#8195;
  不断行的空白格&nbsp;或&#160;

换行

markdown语法中，在编辑时，文字中间空了一行，显示在网页上就是这两段文字被一行隔开
但有时，我们不需要两段文字有那么大的空隙，仅仅只是需要重起一行，有几种方法可以实现：

  1. 当想要重起一行时，按下两个空格，再按Enter
  2. 使用 <br>标签

注：我这里的br标签使用了转义符&#60;和&#62;，分别显示小于号和大于号

黑体与斜体

对个别字进行强调时需要用到黑体或者斜体
**黑体** : 想要某部分字显示黑体效果，用两个*把那部分字体包裹起来
*斜体* : 同样地，斜体效果是用一个*将需要施加斜体效果的字包裹起来

代码注释

单行注释

单行代码注释用 ´这个字符不知道怎么转义，因此无法显示出来，就在键盘数字键‘1’左边，Tab键上面。

多行注释

多行代码注释也用单行注释的那个键，敲三次，紧接着写下编程语言类别，比如C、python，然后换行，写代码段，代码写完后，再敲三次，把整个代码段包裹起来。
