ubuntu安装的相关问题

Author: Vicebery Date: August 2, 2018  16:19:52 Category: 学习记录

以下介绍用U盘安装ubuntu与windows共存的一些问题：

确认安装的位置：

   想要ubuntu和windows共存，就要为ubuntu单独预留一份空间，如果你本身的系统是windows，可以在磁盘管理下，选定某个盘符，点击右键压缩卷，压缩出来的空间作为ubuntu安装的存储空间，大小按个人需求来定，像我学习fabric等区块连平台，预留了50G。压缩卷后看到有一块变成绿色或者灰色，即可用或者未分配空间，该空间就是等会安装ubuntu的位置。

制作U盘启动盘：

   在此步骤之前，确保安装的系统镜像已经下载好了，去ubuntu中文官方网站即可下载最新的版本18.04，我目前用的就是这个版本。然后还需要一个不小于4G的U盘。

   制作启动盘的软件我选用的是powerISO，去官网下载，安装选择试用。

   软件打开后，菜单栏点击“打开”，选择你下载好的系统镜像，然后选择制作U盘启动盘，写入模式建议选择USB-HDD，点击“写入”，一般需要15min，看到刻录完成即成功。

重启系统进入BIOS，选择U盘启动：

   每种型号的计算机进入BIOS的方式一般不一样，thinkpad的笔记本启动后看到屏幕下方有提示，按Ehter键进入选择菜单，然后可以看到最后一行英文意思是选择一种临时启动方式，正是我们所需要的，这样就不用依照网上有些教程所说的设置启动优先级，安装完了还要修改回来，我们只需要临时一次就可以安装好。选择了U盘启动后，就会启动U盘里的系统，等一会儿，出现一个界面，让你选择是试用ubuntu还是安装ubuntu，选择安装。

安装方式的选择：

   按照提示设置，安装时提示你联网，选择不联网，选择不要更新软件，这样可以节省很多时间。

   其中有一个设置界面是选择安装方式，第一个选择是与windows共存，第二个和第三个会覆盖wildows系统–意思是所有磁盘数据会清空，给你安装一个崭新的ubuntu，还有最后一个选项是其他，网上教程一般让你选择其他，如果选择其他，则需要自己分区，提前要想好根目录、/home、/boot、swap分区的大小

   如果选择第一种，则会自行分区，并且不需要选择安装位置，它自行寻找一个空的盘符，就是我们在第一步中已经预留好的空间，/root即根目录会和你预留的安装空间差不多大，/home在根目录下.
